Go语言中输入输出主要依靠标准库"fmt"完成
输出：`fmt.Print`、`fmt.Println`、`fmt.Printf`
输入：`fmt.Scan`、`fmt.Scanln`、`fmt.Scanf`
高效输入输出：`bufio`
文件输入输出：`os`
# 标准输出
`fmt.Print`输出无换行
`fmt.Println`输出后自动换行，输出多参数时会自动添加空格
`fmt.Printf`按照指定的格式输出
## 常用格式化占位符

| 占位符    | 含义          | 示例             |
| ------ | ----------- | -------------- |
| `%v`   | 默认格式输出      | `10`、`[1 2 3]` |
| `%+v`  | 输出更详细的信息    | 结构体显示字段名       |
| `%#v`  | 按 Go 语法格式输出 | `[]int{1, 2}`  |
| `%T`   | 输出变量类型      | `int`、`string` |
| `%d`   | 十进制整数       | `10`           |
| `%b`   | 二进制整数       | `1010`         |
| `%o`   | 八进制整数       | `12`           |
| `%x`   | 十六进制，小写     | `ff`           |
| `%X`   | 十六进制，大写     | `FF`           |
| `%f`   | 浮点数         | `3.140000`     |
| `%.2f` | 保留两位小数      | `3.14`         |
| `%s`   | 字符串         | `hello`        |
| `%q`   | 带引号的字符串或字符  | `"hello"`      |
| `%c`   | 字符          | `A`            |
| `%t`   | 布尔值         | `true`         |
| `%p`   | 指针地址        | `0x...`        |
| `%%`   | 输出百分号       | `%`            |
# 标准输入
`fmt.Scan()`输入按照空格或者换行划分，`fmt.Scan(&name,&age)`
`fmt.Scanln()`按照行划分，`fmt.Scanln(&name,&age)`
`fmt.Scanf()`按照指定格式划分，`fmt.Scanf("%s %d",&name,&age)`
输入函数有两个返回值
`n, err := fmt.Scan(&name, &age)`，一个是成功读取了几个数据，一个是读取过程是否出现错误
